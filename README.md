🛒 AdminJS eCommerce Backend
A secure, high-performance Role-Based eCommerce Admin Dashboard built with AdminJS, Sequelize, and PostgreSQL. This system provides a professional administrative interface to manage users, products, categories, and orders with granular access control.

🚀 Features
Role-Based Access Control (RBAC):

Admin: Full access to all resources, system settings, and global analytics.

User: Restricted access to view personal profile and order history only.

Custom Dashboard:

Dynamic statistics for Admins (Total Revenue, Customer Count, Orders).

Personalized greeting and order summary for Regular Users.

Advanced Order Management: Automatic price calculation and status tracking.

Custom Settings Page: Dedicated UI for managing system configurations.

Security: JWT-based authentication and automatic password hashing/masking.

🛠️ Tech Stack
Backend Framework: Node.js, Express

Admin UI: AdminJS

Database: PostgreSQL with Sequelize ORM

Authentication: JWT (JSON Web Tokens) & bcrypt

Frontend Components: React (Custom AdminJS Components)
